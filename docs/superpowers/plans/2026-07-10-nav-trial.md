# nav-trial Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A TurtleBot 3 Burger in Gazebo Harmonic (headless, Docker on macOS) builds its own map with slam_toolbox, then Nav2 navigates it to programmatic goals — proven by a one-command headless smoke test.

**Architecture:** One Docker image (`ros:jazzy-ros-base-noble` + apt nav stack), one colcon package (`nav_trial_bringup`) holding launch files/params/maps/scripts, one compose service per scenario profile (`sim`/`slam`/`nav`/`test`). All ROS nodes of a scenario run inside a single container via one `ros2 launch` — deliberately bundled because DDS multicast does not cross Docker's bridge network and `network_mode: host` is not real on Docker Desktop/macOS (robium `integration` skill gotcha); single-host sim demo = one supervisable unit per scenario.

**Tech Stack:** ROS 2 Jazzy, Nav2 (+ nav2_simple_commander), slam_toolbox, Gazebo Harmonic via ros_gz, turtlebot3_gazebo 2.3.x, foxglove_bridge, Docker Compose profiles.

## Global Constraints

- Simulation only; no real robot (spec).
- Everything runs in Docker, native `linux/arm64`, no emulation (brief §5).
- Headless: no X anywhere; `gz sim -s -r --headless-rendering`; viz is browser Foxglove via WebSocket :8765 (brief §5).
- ROS 2 **Jazzy** (NOT Lyrical — Nav2 has no Lyrical binaries, navigation2#6123) paired with Gazebo **Harmonic** (brief §2).
- `TURTLEBOT3_MODEL=burger`, `ROS_DOMAIN_ID=42` in every container (integration skill: explicit domain ID).
- `use_sim_time: true` on EVERY node, no exceptions (nav2 skill key directive).
- `rosdep install --from-paths src -y --ignore-src` before every colcon build (ros2 skill key directive).
- Never `pip install` into system Python; this app needs no Python deps beyond apt ROS packages, so no uv layer (environments skill directive, stated deviation).
- The saved map (`map.pgm` + `map.yaml`) is committed to the repo (brief §6).
- Capture robium-skill learnings in `learnings/2026-07-10.md` at the moment they happen (CLAUDE.md — applies to the executor of every task below: stale command, missing guidance, or worked-as-documented ✓, all get a bullet).
- Repo root for all paths below: `apps/nav-trial/` inside `robium-applications`. Commits go to `main` (dedicated proving-ground repo).

## Known-risk playbook (from brief §8 — read before Tasks 3, 6, 7)

- **R1 headless lidar:** first verification of Task 3 is `/scan` publishing. Fallbacks in order: ensure `--headless-rendering` flag present → check OGRE2/EGL errors in gz console output → `apt install mesa-utils libegl1` etc. per error → reduce lidar samples/rate in a copied world file.
- **R2 slow RTF:** measure at Task 3 (`gz stats`); smoke-test timeout in Task 7 is sized as `expected_sim_seconds / measured_RTF * 2`.
- **R3 cmd_vel stamping:** Task 6 has an explicit alignment step; do not skip it.

```dockerfile
FROM ros:jazzy-ros-base-noble

ENV ROS_DISTRO=jazzy

# Nav stack + TB3 sim + SLAM + viz bridge + Gazebo Harmonic (via ros_gz).
RUN apt-get update && apt-get install -y --no-install-recommends \
      ros-jazzy-navigation2 \

      ros-jazzy-turtlebot3-navigation2 \

    && rm -rf /var/lib/apt/lists/*

WORKDIR /ws
COPY src ./src
COPY tests ./tests
COPY scripts ./scripts

# rosdep before build, every time (ros2 skill key directive).
RUN . /opt/ros/${ROS_DISTRO}/setup.sh \
 && apt-get update \
 && rosdep update --rosdistro ${ROS_DISTRO} \
 && rosdep install --from-paths src -y --ignore-src \
 && colcon build --symlink-install \
 && rm -rf /var/lib/apt/lists/*

COPY docker/entrypoint.sh /entrypoint.sh
RUN chmod +x /entrypoint.sh /ws/scripts/*.sh 2>/dev/null || chmod +x /entrypoint.sh

ENV TURTLEBOT3_MODEL=burger
EXPOSE 8765
ENTRYPOINT ["/entrypoint.sh"]
CMD ["bash"]
```

```python
import os
from ament_index_python.packages import get_package_share_directory
from launch import LaunchDescription
from launch.actions import ExecuteProcess
from launch_ros.actions import Node


def generate_launch_description():
    tb3_gazebo = get_package_share_directory('turtlebot3_gazebo')
    world = os.path.join(tb3_gazebo, 'worlds', 'turtlebot3_world.world')          # verify name
    model = os.path.join(tb3_gazebo, 'models', 'turtlebot3_burger', 'model.sdf')  # verify name
    bridge_cfg = os.path.join(tb3_gazebo, 'params', 'turtlebot3_burger_bridge.yaml')  # verify name
    urdf = os.path.join(get_package_share_directory('turtlebot3_description'),
                        'urdf', 'turtlebot3_burger.urdf')                          # verify name

    gz = ExecuteProcess(
        cmd=['gz', 'sim', '-s', '-r', '--headless-rendering', world],
        additional_env={'GZ_SIM_RESOURCE_PATH': os.path.join(tb3_gazebo, 'models')},
        output='screen')
    spawn = Node(package='ros_gz_sim', executable='create',
                 arguments=['-name', 'burger', '-file', model,
                            '-x', '-2.0', '-y', '-0.5', '-z', '0.01'],
                 output='screen')
    bridge = Node(package='ros_gz_bridge', executable='parameter_bridge',
                  parameters=[{'config_file': bridge_cfg, 'use_sim_time': True}],
                  output='screen')
    with open(urdf) as f:
        robot_description = f.read()
    rsp = Node(package='robot_state_publisher', executable='robot_state_publisher',
               parameters=[{'use_sim_time': True, 'robot_description': robot_description}],
               output='screen')
    foxglove = Node(package='foxglove_bridge', executable='foxglove_bridge',
                    parameters=[{'port': 8765, 'use_sim_time': True}],
                    output='screen')
    return LaunchDescription([gz, spawn, bridge, rsp, foxglove])
```

- [ ] **Step 3: Write the failing check**

`tests/check_scan.py`:

```python
#!/usr/bin/env python3
"""Exit 0 iff /scan publishes at least one non-empty LaserScan within TIMEOUT."""
import os
import sys

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import LaserScan

TIMEOUT_S = float(os.environ.get('CHECK_SCAN_TIMEOUT', '90'))


class ScanCheck(Node):
    def __init__(self):
        super().__init__('check_scan')
        self.ok = False
        self.create_subscription(LaserScan, '/scan', self._cb, 10)

    def _cb(self, msg):
        if len(msg.ranges) > 0:
            self.ok = True


def main():
    rclpy.init()
    node = ScanCheck()
    end = node.get_clock().now().nanoseconds / 1e9 + TIMEOUT_S
    while rclpy.ok() and not node.ok and (node.get_clock().now().nanoseconds / 1e9) < end:
        rclpy.spin_once(node, timeout_sec=1.0)
    print('PASS: /scan publishing' if node.ok else 'FAIL: no /scan within timeout')
    sys.exit(0 if node.ok else 1)


if __name__ == '__main__':
    main()
```

`nav_trial_bringup/drive_mapping_route.py` — commander-driven so obstacles are avoided while mapping (waypoints form a ring between the pillar grid and the outer wall of `turtlebot3_world`; TUNE from the Foxglove map view if any goal fails):

```python
#!/usr/bin/env python3
"""Drive an exploration ring around turtlebot3_world under SLAM, then save the map."""
import math
import subprocess
import sys

import rclpy
from geometry_msgs.msg import PoseStamped
from nav2_simple_commander.robot_navigator import BasicNavigator, TaskResult

RING = [  # (x, y) — octagon r~1.7m around the origin, between pillars and wall
    (-1.7, 0.0), (-1.2, 1.2), (0.0, 1.7), (1.2, 1.2),
    (1.7, 0.0), (1.2, -1.2), (0.0, -1.7), (-1.2, -1.2), (-1.7, 0.0),
]
MAP_OUT = '/ws/maps/map'


def make_pose(nav, x, y):
    p = PoseStamped()
    p.header.frame_id = 'map'
    p.header.stamp = nav.get_clock().now().to_msg()
    p.pose.position.x = float(x)
    p.pose.position.y = float(y)
    p.pose.orientation.w = 1.0
    return p


def main():
    rclpy.init()
    nav = BasicNavigator()
    # SLAM provides map->odom; no AMCL, no initial pose needed.
    nav.waitUntilNav2Active(localizer='slam_toolbox')

    reached = 0
    for i, (x, y) in enumerate(RING):
        nav.goToPose(make_pose(nav, x, y))
        while not nav.isTaskComplete():
            rclpy.spin_once(nav, timeout_sec=1.0)
        result = nav.getResult()
        print(f'waypoint {i} ({x},{y}): {result}', flush=True)
        if result == TaskResult.SUCCEEDED:
            reached += 1

    if reached < len(RING) - 2:  # tolerate a couple of failed waypoints
        print(f'FAIL: only {reached}/{len(RING)} waypoints reached', flush=True)
        sys.exit(1)

    print('saving map...', flush=True)
    save = subprocess.run(
        ['ros2', 'run', 'nav2_map_server', 'map_saver_cli', '-f', MAP_OUT,
         '--ros-args', '-p', 'use_sim_time:=true'],
        timeout=60)
    sys.exit(save.returncode)


if __name__ == '__main__':
    main()
```

```python
#!/usr/bin/env python3
"""Map sanity: yaml+pgm exist, and the map has meaningful occupied AND free area."""
import re
import sys


def read_pgm(path):
    with open(path, 'rb') as f:
        data = f.read()
    if not data.startswith(b'P5'):
        sys.exit(f'FAIL: {path} is not a binary PGM (P5)')
    # strip comments, parse header: magic, width, height, maxval
    tokens = []
    for line in data.split(b'\n'):
        if line.startswith(b'#'):
            continue
        tokens.extend(line.split())
        if len(tokens) >= 4:
            break
    width, height, maxval = int(tokens[1]), int(tokens[2]), int(tokens[3])
    header_len = data.find(bytes(tokens[3])) + len(bytes(tokens[3])) + 1
    pixels = data[header_len:header_len + width * height]
    return width, height, maxval, pixels


def main():
    if len(sys.argv) != 2:
        sys.exit('usage: check_map.py <map.yaml>')
    yaml_path = sys.argv[1]
    text = open(yaml_path).read()
    m = re.search(r'image:\s*(\S+)', text)
    if not m:
        sys.exit('FAIL: no image key in map yaml')
    import os
    pgm_path = os.path.join(os.path.dirname(yaml_path), m.group(1))

    width, height, maxval, pixels = read_pgm(pgm_path)
    total = width * height
    # ROS map_saver default thresholds: free >= 0.75*maxval (white-ish),
    # occupied <= 0.35*maxval (dark). Unknown is the middle gray.
    free = sum(1 for p in pixels if p >= 0.75 * maxval)
    occ = sum(1 for p in pixels if p <= 0.35 * maxval)

    print(f'{width}x{height} free={free} occ={occ} total={total}')
    if total < 10_000:
        sys.exit('FAIL: map suspiciously small')
    if free < 0.05 * total:
        sys.exit('FAIL: almost no free space mapped')
    if occ < 100:
        sys.exit('FAIL: almost no obstacles mapped (walls/pillars missing)')
    print('PASS: map is non-trivial')


if __name__ == '__main__':
    main()
```

`nav_trial_bringup/send_goals.py`:

```python
#!/usr/bin/env python3
"""Send nav goals on the saved map; exit 0 iff all SUCCEEDED within --timeout."""
import argparse
import sys
import time

import rclpy
from geometry_msgs.msg import PoseStamped
from nav2_simple_commander.robot_navigator import BasicNavigator, TaskResult

SPAWN = (-2.0, -0.5)  # matches sim.launch.py spawn pose
DEFAULT_GOALS = '1.5,0.0;-1.7,1.2'


def make_pose(nav, x, y):
    p = PoseStamped()
    p.header.frame_id = 'map'
    p.header.stamp = nav.get_clock().now().to_msg()
    p.pose.position.x = float(x)
    p.pose.position.y = float(y)
    p.pose.orientation.w = 1.0
    return p


def main(argv=None):
    ap = argparse.ArgumentParser()
    ap.add_argument('--goals', default=DEFAULT_GOALS,
                    help='semicolon-separated x,y pairs in map frame')
    ap.add_argument('--timeout', type=float, default=300.0,
                    help='wall-clock seconds for the WHOLE run')
    args = ap.parse_args(argv)
    goals = [tuple(map(float, g.split(','))) for g in args.goals.split(';')]

    rclpy.init()
    nav = BasicNavigator()
    nav.setInitialPose(make_pose(nav, *SPAWN))   # AMCL is silent without this
    nav.waitUntilNav2Active()

    deadline = time.monotonic() + args.timeout
    for i, (x, y) in enumerate(goals):
        nav.goToPose(make_pose(nav, x, y))
        while not nav.isTaskComplete():
            if time.monotonic() > deadline:
                print(f'FAIL: timeout during goal {i} ({x},{y})', flush=True)
                nav.cancelTask()
                return 1
            rclpy.spin_once(nav, timeout_sec=1.0)
        result = nav.getResult()
        print(f'goal {i} ({x},{y}): {result}', flush=True)
        if result != TaskResult.SUCCEEDED:
            print('FAIL: goal not reached', flush=True)
            return 1

    print('PASS: all goals reached', flush=True)
    return 0


if __name__ == '__main__':
    sys.exit(main())
```

(Goal coordinates are on the SLAM map from Task 5 — pick two clearly free cells by inspecting the map in Foxglove; the defaults above assume the standard turtlebot3_world layout. Update `DEFAULT_GOALS` if the map says otherwise.)
