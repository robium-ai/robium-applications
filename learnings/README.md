# learnings/

Dated friction notes from building apps with the robium plugin — the raw input to
robium's skill-hardening loop. See the root README's "Learnings loop" section for the
format. One file per day per app: `YYYY-MM-DD.md` (add `-<app>` suffix if two apps run
the same day). Move files to `absorbed/` once a robium hardening session has folded them
into skill edits.
